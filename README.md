# Dynamic Web Application

A simple web application built using Flask (Python), JavaScript, HTML, and CSS that interacts with user input and displays dynamic results.

## Features
- Accepts user input from the frontend
- Processes data using Flask backend
- Displays results dynamically using JavaScript
- Enabled CORS for smooth communication between frontend and backend

## Tech Stack
- Python (Flask)
- JavaScript
- HTML
- CSS

## Setup & Run

1. Clone the repository:
   git clone https://github.com/your-username/your-repo-name.git

2. Navigate to the folder:
   cd your-repo-name

3. Install dependencies:
   pip install flask flask-cors

4. Run the app:
   python app.py


## Project Structure
- app.py → Backend (Flask)
- templates/ → HTML files
- static/ → CSS & JavaScript
